About Me

Ketertarikan saya pada dunia teknologi dimulai sejak bangku SMK, tepat saat pertama kali saya mencoba memahami bagaimana komputer bisa saling berkomunikasi dengan komputer lain melalui jaringan. Pemahaman ini dibuktikan secara nyata saat saya menjalani Praktek Kerja Lapangan (PKL) di PT Carano Integrasi Teknologi.

Di sana, saya tidak hanya belajar secara teori, tetapi terjun langsung ke lapangan. Mulai dari instalasi kabel LAN, monitoring jaringan, hingga mengkonfigurasi EoIP Tunnel pada MikroTik. Memasuki dunia perkuliahan, saya menyadari bahwa memahami infrastruktur saja tidak cukup, saya juga ingin menciptakan sistem yang berjalan di atasnya.

Hal ini mendorong saya untuk mulai mempelajari alur pengembangan perangkat lunak, salah satunya melalui pembuatan program pengelola barang sederhana menggunakan Golang. Saat ini, saya sedang mempelajari perancangan website (UI/UX) dan aktif menulis di blog mahasiswa.
